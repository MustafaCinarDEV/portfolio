### 👋 Personal Portfolio

Nuxt JS is a developer portfolio built with the vue js framework.


### 🔧 Getting  Started
 - First of all, you should choose `git` to clone the project into your file. 

 - Download the project;
     - `npm i`
 - Run the project in dev mode
     - `npm run dev`
